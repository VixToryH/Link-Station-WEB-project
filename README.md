# Link-Station

Веб-застосунок для моделювання системи продажу залізничних квитків.

## Технології
- React 18
- Vite
- React Router DOM
- React Toastify

## Функціонал
- Перегляд списку потягів та рейсів
- Пошук за маршрутом або номером потяга
- Вибір вагона та місць
- Бронювання квитків з валідацією
- Збереження бронювань у localStorage
